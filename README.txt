-------------------------English Version----------------------------------
This is a Grammar Analysis Program written in Java, using LR(1) method of grammatical analysis.
It can judge whether the input grammmar is valid or not.

-------------------------------注意事项-----------------------------
0.本语法分析器基于LR(1)进行设计，原工程见文件夹“SwarmSyntax”，打包后名为“SwarmSyntax.jar”，若有JAVA环境即可运行。

1.默认的开始符号即为S，请注意，因此文法中开始符号必须为S,另外文法中也不能再出现 S'->S，若有，请删除该产生式。

2.文法产生式中不能出现|（或的符号），也即应该写成两个或以上的独立产生式！

3.文法产生式中的“空”请用$表示！

4.生成的项目集规范族和正常的计算结果可能顺序不一样（这是由于当时设计时我采用的策略问题），
由此导致分析过程的ACTION 和 GOTO表的内容和手算不一样，但是结果肯定是正确的！

5.附有两个文法产生式，分别名叫 Grammar—Syntax_test 和 Grammar—Syntax_test2
---------------------------------------------------------------------

-------------------------------使用说明-----------------------------
1.先点击Open File按钮，打开一个符合标准的文法产生式的文本文档（附有两个文法产生式）
*注意：必须从open file中打开文法，不能直接复制到Input Area中！！！（否则会出错）*

2.在中间的单行输入框中输入待判定的句子

3.点击Analyse按钮则进行分析，会在中部右边的“Output the standar family of itemsets”中
显示分析出的项目集规范族！在下面的“Output the procedure of the analysing”中输出分析过程！

4.右上角的Result也显示出分析结果，若符合文法，则显示为Yes!!，若不符合，则显示为No!!

5.点击Clear All清空所有内容

6.直接点击Open File（不点击Clear All亦可）可再次打开其他的文法产生式，进行新一轮的分析！
